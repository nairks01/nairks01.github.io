COVID Case-Fatality RatioVisualizer

A Java Program depicting CFR data based on race in different states

For my Software Design & Data Structures (CS 2114) class, I had to make a program which outputted a graph depicting Case-Fatality Ratio (CFR) data based on race in different states. I worked with 2 other classmates to complete this project. The input data was provided to us as .csv files.

This project helped me become a confident Java programmer because it taught me various concepts such as data structures, sorting algorithms, I/O, and how to implement GUIs.
