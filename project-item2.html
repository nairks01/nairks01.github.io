MIPS32 Assembler

An assembler written in C which supports the MIPS32 instruction set

For my Computer Organization II (CS 2506) class, I had to make an assembler which supported the MIPS32 instruction set. This was one of the most time-consuming and challenging projects I've worked on as it required extensive knowledge about the MIPS instruction set, the MIPS datapath, I/O in C, data structures, and how assemblers work.

This project helped me become a confident C programmer, expanded my understadning of computer architecture, and made me comfortbale with using the GCC compiler, GDB Debugger, and Valgrind.
